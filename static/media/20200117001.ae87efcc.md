[React Native](https://facebook.github.io/react-native/docs/running-on-device) 블로그에 있는 내용을 정리한다.

### 개발환경

> Windows 10<br>
> React Native CLI<br>
> React Native 0.61
> Android Device<br>
> Android 10
> Visual Studio Code

> Note
>
> > PC와 안드로이드 device를 USB케이블로 연결한후 작업을 시작하고
> > Wi-fi 디버깅 세팅이 모두 완료되면 USB케이블을 해제한다.

1. 안드로이드 디바이스를 USB케이블을 이용하여 PC에 연결 후 VSC툴에서 npx react-native run-android 를 실행하여 app을 디바이스에 띄운다.
   ![](../../../images/20200117001_6.png)

2. 안드로이드 디바이스를 들고 두세번 정도 좌우로 흔들어 "개발자메뉴"가 나오면 "Settings" 메뉴로 이동한다.
   ![](../../../images/20200117001_1.png)

3. Debug server host & port for device 메뉴로 이동한다.
   ![](../../../images/20200117001_2.png)

4. device와 연결되어있는 PC IP와 8081포트를 입력한다.
   ![](../../../images/20200117001_4.png)

5. 입력후 뒤로가기 버튼을 클릭하여 개발자메뉴로 다시 돌아온다.
   뒤로가기 버튼을 클릭하면 개발자메뉴가 사라진다.
   디바이스를 두세번좌우로 흔들어서 다시 개발자 메뉴로 들어온다.

6. Reload 버튼을 클릭하여 새로고침을 한다.
   ![](../../../images/20200117001_5.png)

> Note
>
> > 만약 Reload버튼시 오류가 발생한다면 PC의 방화벽 정책을 확인해보길 바란다.

커피숍이나 다른 외부에서 WI-IF를 접속하면 공용네트워크로 접속하기 때문에 개인네트워크로 설정해놓은 방화벽정책은 쓸수가 없다.
그래서 공용을 개인네트워크로 변경을 해주던지 아니면 공용네트워크 방화벽정책을 수정해주면 오류를 잡을 수 있다.
