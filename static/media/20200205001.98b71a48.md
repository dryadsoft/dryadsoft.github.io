## javascript 파일 컴파일 하기

> node.js의 javascript 한가지만 간단하게 컴파일하는 방법을 알아보자.<br/>
> gulp개발환경 세팅이 아직 안되어 있으면 [이전포스트](https://dryadsoft.github.io/posts/gulp/1)를 참고하길 바란다.

### src/js 폴더에 *javascript*파일을 생성 후 테스트 코드를 작성한다.

```javascript
// src / js / util.js;
const plus = (a, b) => a + b;

export { plus };
```

```javascript
// src / js / index.js;
import { plus } from "./util";

plus(4, 5);
```

### javascript를 컴파일하기 위하여 필요한 모듈을 설치한다.

```shell
> yarn add gulp-babel -D
```

### gulpfile.babel.js 파일에 아래 내용을 작성한다.

```javascript
import gulp from "gulp";
import babel from "gulp-babel";

const routes = {
  js: {
    src: "src/**/*.js", // 대상 javascript파일
    dest: "build" // javascript파일이 컴파일되어서 생성되는 폴더
  }
};

gulp.task("dev", () => {
  // package.json의 scripts에 작성한 "gulp dev" task
  return gulp
    .src(routes.src) // 출발지
    .pipe(babel()) // babel을 이용하여 컴파일
    .pipe(gulp.dest(routes.js.dest)); // 목적지
});
```

### gulp 실행

1. gulp를 실행하면 아래와같이 정상적으로 실행된걸 확인 할 수 있다.

```shell
$ yarn dev
yarn run v1.21.0
$ gulp dev

[07:20:20] Starting 'dev'...
[07:20:20] Finished 'dev' after 58 ms

```

> 실행결과 root디렉토리에 build 폴더가 생성된다.<br/>
> 생성된build 폴더 안의 구조는 src폴더의 구조와 똑같다.<br/>
> build/js 폴더 안에 컴파일된 소스를 확인 할 수 있다.
