### 프로젝트 git 생성

```shell
git init
```

### 새 리포지토리 remote 추가

```shell
git remote add origin 리포지토리url
```

### 기존 리포지토리 remote 제거

```shell
git remote remove origin
```

### branch 생성

```shell
git push origin 브랜치명
```

### branch 삭제

```shell
git push origin --delete 브랜치명
```
