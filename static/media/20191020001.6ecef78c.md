## gulp 란 무엇인가?

```
gulp는 흔히 Task Runner로 알려져있다..
많은 일을 Task로 만들고 그 Task를 실행하면
그결과물을 자동으로 생성해준다.
```

### Task 란?

1. 자동 이미지압축
2. 최신자바스크립트를 구형브라우저에서 이해 할 수 있도록 컴파일.

위 1번, 2번과 같은 일을 Task로 만들어서 사용한다.

### gulp 개발환경설정

1. gulp 전역 설치

- yarn 사용

```shell
  - yarn global add gulp-cli
```

- npm

```shell
  - npm install gulp-cli -g
```
