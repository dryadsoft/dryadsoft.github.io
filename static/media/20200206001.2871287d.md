이번 강좌는 nodejs를 typescript를 이용하여 작성하는 방법을 알아보겠다.
기본적인 gulp개발환경설정은 [이전포스트](./1)에서 확인 할 수 있다.

## typescript 모듈 설치

```shell
> yarn add typescript gulp gulp-typescript -D
```

## typescript 예제파일 작성

> src폴더안에 tsUtil.ts 파일을 생성한다.

```typescript
const hello = (message: string): void => console.log(`hello ${message}`);

export default hello;
```

> src폴더안에 main.ts 파일을 생성한다.

```typescript
import hello from "./tsUtil";

function aplus(a: number, b: number): void {
  console.log(a + b);
}
hello("typescript");
aplus(1, 2);
```

## root디렉토리에 tsconfig.json파일 생성

```json
  "include": ["src"],
  "compilerOptions": {
    "noImplicitAny": true,
    "target": "es5"
  }
```

## gulpfile.babel.js 파일 작성

```javascript
import gulp from "gulp";
import babel from "gulp-babel";
import ts from "gulp-typescript"; // typescript 추가
const tsProject = ts.createProject("tsconfig.json"); // typescript 추가

const routes = {
  js: {
    src: "src/**/*.js",
    dest: "build"
  },
  // typescript 추가
  typescript: {
    dest: "build"
  }
};

const js = () =>
  gulp
    .src(routes.js.src)
    .pipe(babel())
    .pipe(gulp.dest(routes.js.dest));

// typescript 추가
const typescript = () =>
  tsProject
    .src()
    .pipe(tsProject())
    .js.pipe(gulp.dest(routes.typescript.dest));

export const dev = gulp.series([js, typescript]); // js와 typescript 순차적으로 컴파일 진행
```

## gulp 실행

1. gulp를 실행하면 아래와 같이 js가 먼저 실행된 후 typescript가 정상적으로 실행된걸 확인 할 수 있다.

```shell
$ yarn dev
```

> 실행결과 root디렉토리의 지정한 build폴더에 컴파일된 파일이 아래와 같이 node에서 실행할 수 있는 js파일로 변환되어 생성된다.

```javascript
// tsUtil.ts -> tsUtil.js
"use strict";
Object.defineProperty(exports, "__esModule", { value: true });
var hello = function(message) {
  return console.log("hello " + message);
};
exports.default = hello;
```

```javascript
// main.ts -> main.js
"use strict";
Object.defineProperty(exports, "__esModule", { value: true });
var tsUtil_1 = require("./tsUtil");
function aplus(a, b) {
  console.log(a + b);
}
tsUtil_1.default("typescript");
aplus(1, 2);
```

## node 실행하기

> 실행은 main.ts파일이 아닌 컴파일되어 생성된 main.js파일을 실행한다.

```shell
$ node build/main.js
hello typescript
3
```
